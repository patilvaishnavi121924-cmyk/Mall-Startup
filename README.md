# 🏬 Mall Startup

Mall Startup is a responsive single-page shopping mall website created using HTML, CSS, Bootstrap 5, and JavaScript. The project showcases a modern mall landing page with attractive sections such as About, Services, Gallery, Membership Plans, Customer Reviews, FAQ, Blogs, and Contact.

## 🚀 Features

- Responsive Navigation Bar
- Hero Banner
- About Section
- Services Section
- Mall Gallery
- Statistics Section
- Customer Reviews
- Membership Plans
- FAQ Accordion
- Latest Blogs
- Contact Form
- Dark Mode Toggle
- Bootstrap Icons
- Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
